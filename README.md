Nautobot plugin to support any kind of attachments in Nautobot objects

### Installation
- clone repo
- `cd nautobot-jinja-filters`
- activate Nautobot's venv
- `pip install .`
- add `nautobot_jinja_filters` in the PLUGINS section of your `nautobot_config.py`
